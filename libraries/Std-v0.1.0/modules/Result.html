Std modules

Std.Result

The canonical recoverable-error datatype for Hiko. Prefer Result over ad hoc Either when the branches mean success vs failure.

Use

import Std.Result

Raw source: Result.hml

Datatype

datatype ('a, 'e) result = Ok of 'a | Err of 'e

Public surface

Pipelined example

import Std.Result

val loaded =
  Result.Ok "42"
  |> Result.and_then string_to_int
  |> Result.map (fn n => n + 1)

val answer =
  loaded
  |> Result.unwrap_or 0

val flattened =
  Result.Ok (Result.Ok 42)
  |> Result.flatten
