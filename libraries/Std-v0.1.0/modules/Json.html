Std modules

Std.Json

The tagged JSON datatype used by the JSON parsing builtins and typed JSON examples.

Use

import Std.Json

Raw source: Json.hml

Datatype

datatype json =
    JNull
  | JBool of bool
  | JInt of int
  | JFloat of float
  | JStr of string
  | JArray of json list
  | JObject of (string * json) list

Parse example

A concrete JSON string parsed through the runtime builtin:

import Std.Json

val input_json =
  "{\"count\":3,\"ok\":true,\"tags\":[\"ml\",\"effects\"]}"

val parsed = json_parse input_json

The resulting value has the same shape as:

import Std.Json

val parsed_shape =
  Json.JObject [
    ("count", Json.JInt 3),
    ("ok", Json.JBool true),
    ("tags", Json.JArray [Json.JStr "ml", Json.JStr "effects"])
  ]

Exhaustive constructor example

import Std.Json

val null_value = Json.JNull
val bool_value = Json.JBool true
val int_value = Json.JInt 3
val float_value = Json.JFloat 3.14
val string_value = Json.JStr "hello"
val array_value = Json.JArray [Json.JInt 1, Json.JInt 2]
val object_value =
  Json.JObject
    [("ok", Json.JBool true), ("count", Json.JInt 3)]

fun describe value =
  case value of
      Json.JNull => "null"
    | Json.JBool b => if b then "bool:true" else "bool:false"
    | Json.JInt n => "int:" ^ int_to_string n
    | Json.JFloat _ => "float"
    | Json.JStr s => "str:" ^ s
    | Json.JArray _ => "array"
    | Json.JObject _ => "object"
